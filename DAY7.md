Biggest mistakes to avoid
Using Opus for everything — it's slower and heavier. Save it for truly complex tasks; Sonnet handles 80% of student work better.
Vague prompts like "help me code this" — always share the full context: your language, what you've tried, and the exact error message.
Skipping iterations — don't accept the first output. Ask Claude to improve, simplify, or explain its own answer.
Using High/Max effort for everything — it slows responses. Standard effort is ideal for most daily tasks and just as accurate.
Copy-pasting code without understanding it — ask Claude to explain each part. Learning compounds; blind copying doesn't.
